# DevSpeed ⚡

> How fast is your developer brain?

A fun, fully interactive web app that tests three core developer skills — typing speed, reading comprehension, and focus. Built with zero frameworks, zero dependencies. Pure vanilla HTML, CSS, and JavaScript.

---

## Features

- **Typing Speed** — Real-time WPM counter, accuracy tracker, 3 difficulty levels (Easy / Medium / Hard with real code snippets), 60s timer, and an S/A/B/C rank at the end
- **Reading Speed** — Read a tech passage, get timed, then answer comprehension questions. Shows your reading WPM + quiz score
- **Focus Mode** — 18 words appear on screen, one is highlighted — type only that word as fast as possible. Includes a keystroke heatmap

---

## Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- No libraries, no frameworks, no build tools

---

